# AICO - Asistente Inteligente para Consultorios Odontológicos

**Trabajo Final de Grado - Javier Bonet**

## Descripción del Proyecto

AICO (Asistente Inteligente para Consultorios Odontológicos) es un sistema inteligente diseñado para la gestión eficiente de historias clínicas en consultorios odontológicos. Este proyecto forma parte del Trabajo Final de Grado y presenta una demostración funcional del sistema.

## Estructura del Repositorio

```
aico/
├── src/              # Código fuente de la aplicación DEMO
├── docs/             # Documentación técnica del proyecto
├── anexos/           # Anexos y documentación adicional para evaluación
└── README.md         # Este archivo
```

## Características Principales

- Gestión digital de historias clínicas odontológicas
- Interfaz intuitiva para profesionales de la salud dental
- Sistema de almacenamiento seguro de información de pacientes
- Generación de reportes y estadísticas
- Asistencia inteligente en la toma de decisiones clínicas
